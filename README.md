# homebrew-vo

Homebrew tap for the [Vo programming language](https://github.com/vo-lang/volang).

## Install

```sh
brew tap vo-lang/vo
brew install vo
```

## Usage

```sh
vo run hello.vo
vo check .
vo build .
vo test
vo mod add github.com/user/pkg
vo version
```

## Upgrade

```sh
brew upgrade vo
```
